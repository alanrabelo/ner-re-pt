Acesso à base dados Legis-PALOP
Informação
Os Serviços de Documentação acabam de disponibilizar o acesso à base de dados Legis-PALOP, que consiste numa plataforma de conhecimento e partilha de informação jurídica entre os Países Africanos de Língua Oficial Portuguesa. Através da versão gratuita - http://www.legis-palop.org/bd - é possível consultar e pesquisar a base de dados de legislação. No entanto, o acesso às restantes funcionalidades dos sistema, que incluem a consulta de informação detalhada, requerem as credenciais de autenticação obtidas por subscrição de uma licença. Assim, o acesso à versão integral desta base de dados é apenas disponibilizado nos SD, com prévia inserção das credenciais pelo funcionário.
